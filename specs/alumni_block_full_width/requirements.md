# EARS Requirements: alumni_block_full_width

This specification defines the requirement to stretch the alumni masterclass card (`alumni-masterclass-block`) to the full width of its section container.

---

## 🎯 Requirements Specification

### Feature Scope & Traceability Table

| ID | Type | Condition / Trigger | System Response (Shall...) | Verified By (Test ID) |
| :- | :--- | :------------------ | :------------------------- | :-------------------- |
| **R_ALUMNI_BLOCK_FULL_WIDTH** | Ubiquitous | *None* | The system shall display the alumni masterclass card (`alumni-masterclass-block`) with a maximum width matching the section container width (100% / 960px). | `T_ALUMNI_BLOCK_FULL_WIDTH_VERIFY` |

---
*Requirements are proposed for verification.*
