# Copywriting: Voice & Tone Guidelines

Al generar texto para ProjectPilot, adhiérete a esta personalidad de marca.

## Brand Personality Keywords
* Profesional pero accesible
* Directo y eficiente
* Tecnológico pero sin jerga innecesaria
* Empático

## Grammar & Mechanics rules
* **Headings:** Usa Title Case para encabezados principales (H1, H2). Usa sentence case para subencabezados (H3+).
* **Punctuation:** Evita los signos de exclamación (!) en copys estándar de la interfaz. Usa puntos para oraciones completas.
* **Clarity:** Prefiere la voz activa sobre la pasiva. Mantén las oraciones concisas.

## Terminology Guide

| Do Not Use | Use Instead |
| :--- | :--- |
| "Utilizar" | "Usar" |
| "Con el fin de..." | "Para..." |
| "Aprovechar" | "Usar" |
