# 🧠 AI-Powered Design Intelligence (UI/UX Pro Max)

Este recurso contiene el framework avanzado para generar interfaces modernas, hermosas y de alto rendimiento. Se basa en la metodología UI/UX Pro Max.

## 📋 Dimensiones del Diseño

### 1. PATTERN & LAYOUT (El Esqueleto)
- **SaaS**: Hero + Features + Social Proof + CTA.
- **Micro SaaS**: Minimal & Direct + Live Demo.
- **Analytics**: Bento Grid + Actionable Insights.

### 2. STYLE & AESTHETIC (La Piel)
- **Glassmorphism**: Capas transparentes, desenfoque de fondo.
- **Aurora UI**: Gradientes vibrantes tipo "Northern Lights".
- **Linear/Vercel**: Minimalismo técnico, dark mode, bordes de 1px.
- **Bento Grid**: Modular, denso en información pero limpio.

### 3. COLOR & THEME (La Paleta)
- **Trust (Navy/Slate)**: Para finanzas y salud.
- **Vibrant (Indigo/Emerald)**: Para startups tech.
- **Luxury (Stone/Gold)**: Para productos premium.

### 4. TYPOGRAPHY (La Voz)
- **Modern**: Inter + Roboto + JetBrains Mono.
- **Luxury**: Playfair Display + Montserrat.
- **Brutalist**: Space Grotesk + IBM Plex Sans.

### 5. ANIMATIONS & INTERACTIONS (El Alma)
- **Micro-interacciones**: Scale up (1.02), Lift (-2px), Glow.
- **Scroll**: Fade up, Parallax sutil, Progress indicators.
- **Loading**: Skeleton loaders con efecto shimmer.

## 🚫 Anti-Patrones a Evitar
- No usar animaciones de más de 300ms para interacciones.
- No usar gris claro sobre fondo blanco (accesibilidad).
- No usar labels dentro de los inputs.
- No usar "Click here" (usar texto descriptivo).
